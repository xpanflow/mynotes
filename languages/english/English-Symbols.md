| ,   | comma               | /ˈkɑːmə/                 |
| :   | colon               | /ˈkoʊlən/                |
| ;   | semicolon           | /ˈsemiˌkoʊlən/           |
| ?   | question mark       | /ˈkwestʃən mɑːrk/        |
| !   | exclamation mark    | /ˌekskləˈmeɪʃən mɑːrk/   |
| @   | at sign             | /æt saɪn/                |
| #   | hash / pound sign   | /hæʃ/；/paʊnd saɪn/       |
| $   | dollar sign         | /ˈdɑːlər saɪn/           |
| %   | percent sign        | /pərˈsent saɪn/          |

| < > | angle brackets      | /ˈæŋɡəl ˈbrækɪts/        |

编程里常用说法：

* `.` = dot
* `/` = slash
* `\` = backslash
* `_` = underscore
* `-` = hyphen 或 dash
* `#` = hash
* `@` = at
* `*` = star 或 asterisk
* `{}` = braces
* `[]` = brackets
* `()` = parentheses
