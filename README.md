# my-notes

Personal knowledge base — capture, organize, revisit.

## Structure

| Folder | Purpose |
|--------|---------|
| `inbox/` | Quick capture — triage regularly |
| `daily/` | Journal & daily logs (`YYYY/MM/YYYY-MM-DD.md`) |
| `work/` | Meetings, projects, work ideas |
| `dev/` | Software development notes |
| `devops/` | Infra, CI/CD, cloud |
| `learning/` | General structured learnings & books |
| `languages/` | English & other languages |
| `news/` | News clips & commentary |
| `personal/` | Goals, reflections, personal ideas |
| `family/` | Family events & records |
| `_templates/` | Reusable note templates |

## Workflow

1. **Capture** → drop anything into `inbox/`
2. **Triage** → move to the right folder weekly
3. **Daily** → use `_templates/daily.md` in `daily/YYYY/MM/`
4. **Link** → reference related notes with relative paths

## Naming

- Daily: `2026-05-15.md`
- Topics: `kebab-case-topic.md`
- Meetings: `YYYY-MM-DD-topic.md`
